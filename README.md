# PDF Imposition Stamper (WPF)

Wrapper-приложение на C# для подготовки PDF к печати:
- выбор страниц из исходного PDF;
- проставление номера спуска через Ghostscript;
- размножение страниц по тиражу;
- финальная сборка через qpdf.

## Быстрый старт

1. Положите в папку `Tools/` рядом с проектом:
   - `qpdf.exe` (+ сопутствующие dll, если требуются)
   - `gswin64c.exe`
2. Соберите проект в Visual Studio 2022+.
3. Запустите и выберите исходный PDF.
4. Заполните таблицу: `Номер страницы`, `Количество копий`.
5. Нажмите **СТАРТ**.

## Примечания

- Приложение ожидает, что инструменты доступны в `AppContext.BaseDirectory/Tools`.
- Вставка из Excel поддерживается через `Ctrl+V` в таблице (формат: `page<TAB>copies`).
