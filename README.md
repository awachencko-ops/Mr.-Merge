# PDF Imposition Stamper (WPF)

Wrapper-приложение на C# для подготовки PDF к печати:
- выбор страниц из исходного PDF;
- проставление номера спуска через Ghostscript;
- размножение страниц по тиражу;
- финальная сборка через qpdf.

## Запуск в Visual Studio

1. Установите **Visual Studio 2022** с workload **.NET desktop development**.
2. Откройте решение `MrMergePdfStamper.sln`.
3. В корне проекта положите в папку `Tools/`:
   - `qpdf.exe` (+ сопутствующие DLL, если требуются вашей сборкой qpdf)
   - `gswin64c.exe`
4. Нажмите `F5` (или **Start Debugging**).

> При запуске утилиты копируются в `bin/<Configuration>/net8.0-windows/Tools` автоматически.

## Использование

1. Выберите исходный PDF.
2. Заполните таблицу: `Номер страницы`, `Количество копий`.
3. При необходимости раскройте «Настройки штампа».
4. Нажмите **СТАРТ**.

## Примечания

- Вставка из Excel поддерживается через `Ctrl+V` в таблице (формат: `page<TAB>copies`).
- Числа в настройках штампа можно вводить как через запятую, так и через точку.
